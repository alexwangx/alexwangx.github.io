How to do the Labs

在动手练习中, 您将使用各种Flink API实现Flink程序. 您还将学习如何使用Apache Maven打包Flink程序以及如何在正在运行的Flink实例上执行打包的程序.

以下步骤将指导您完成以下过程: 使用提供的数据流, 实现第一个Flink流程序以及在正在运行的Flink实例上打包和执行程序.

我们假设您已经按照我们的设置指南设置了 开发环境, 并从github获得了 flink-training-exercises 存储库的本地克隆.

1. Learn about the data

最初的练习全部基于有关出租车行程和出租车票价的事件的数据流. 这些流由源函数产生, 源函数从输入文件中读取数据. 请阅读这些 说明以了解如何使用它们.

2. Modify ExerciseBase

修改 com.ververica.flinktraining.exercises.datastream_java.utils.ExerciseBase 里面的路径地址.

public final static String pathToRideData =
    "/Users/david/stuff/flink-training/trainingData/nycTaxiRides.gz";
public final static String pathToFareData =
    "/Users/david/stuff/flink-training/trainingData/nycTaxiFares.gz";

3. Run and debug Flink programs in your IDE

Flink程序可以在IDE中执行和调试. 这极大地简化了开发过程, 并提供了类似于在任何其他Java（或Scala）应用程序上工作的经验.

在IDE中启动Flink程序就像运行其 main() 方法一样容易. 在后台, 执行环境将在同一进程中启动本地Flink实例. 因此, 也可以在代码中放置断点并对其进行调试.

假设您有一个导入了 flink-training-exercises 项目的IDE, 则可以运行（或调试）一个简单的流作业, 如下所示:

  • 打开 com.ververica.flinktraining.examples.datastream_java.basics.RideCount 类

  • 使用您的IDE运行（或调试）RideCountExample 类的 main() 方法.

4. Exercises, Tests, and Solutions

这些练习中的许多练习都包含带有大多数入门必不可少的样板代码的Exercise类, 以及带有在您实施适当的解决方案之前将失败的测试的JUnit Test类, 以及具有完整解决方案的Solution类.

现在, 您准备开始第一个练习.
