Setup your Development Environment

以下说明指导您完成设置开发环境的过程, 以开发, 调试和执行Flink开发人员培训练习和示例的解决方案.

1. Software requirements

Flink支持Linux, OS X和Windows作为Flink程序和本地执行的开发环境. Flink开发设置需要以下软件, 并且应将其安装在系统上:

  • a JDK for Java 8 or Java 11 (不支持其他版本 java)

  • Apache Maven 3.x

  • Git

  • IDE

2. Clone and build the flink-training-exercises project

flink-training-exercises 项目包含用于编程的练习, 测试和参考解决方案, 以及大量示例. 从Github克隆 flink-training-exercises 项目并进行构建.

对于Java, 请使用java分支:

git clone --branch java https://github.com/ververica/flink-training-exercises.git
cd flink-training-exercises
mvn clean package

对于Scala, 请使用master分支:

git clone https://github.com/ververica/flink-training-exercises.git
cd flink-training-exercises
mvn clean package

如果您以前从未这样做过, 那么到此为止, 您将最终下载此Flink培训练习项目的所有依赖项. 这通常需要几分钟, 具体取决于您的Internet连接速度.

3. Import the flink-training-exercises project into your IDE

该项目需要作为Maven项目导入到您的IDE中.

完成后, 您应该可以打开 com.ververica.flinktraining.exercises.datastream_java.basics.RideCleansingTest 并成功运行此测试.

4. Download the data sets

您还需要通过运行以下命令来下载本培训中使用的出租车数据文件.

wget http://training.ververica.com/trainingData/nycTaxiRides.gz
wget http://training.ververica.com/trainingData/nycTaxiFares.gz

使用 wget 或其他工具（例如curl或Chrome）下载这些文件都没有关系, 但是无论如何, 只要获得数据, 请不要解压缩或重命名.gz文件 . 默认情况下, 某些浏览器将执行错误的操作.

注意: 需要修改 com.ververica.flinktraining.exercises.datastream_java.utils.ExerciseBase 中的这些数据文件的硬路径.
