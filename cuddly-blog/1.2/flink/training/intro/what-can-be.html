What can be Streamed?

Flink的Java和Scala的DataStream API可让您流式传输可序列化的所有内容. Flink自己的序列化器用于

  • 基本类型, 即String, Long, Integer, Boolean, Array

  • 复合类型: 元组, POJO 和 Scala case classes

对于其他类型, Flink 会使用Kryo.

Java

Tuples

对于Java, Flink 通过 TUple1 到 Tuple25 类型定义.

Tuple2<String, Integer> person = new Tuple2<>("Fred", 35);

// zero based index!
String name = person.f0;
Integer age = person.f1;

POJOs

POJO（普通的旧Java对象）是: * 有一个空的默认构造函数 * 所有字段都是 public, 或者 有一个默认的getter和setter 方法.

Example:

public class Person {
    public String name;
    public Integer age;
    public Person() {};
    public Person(String name, Integer age) {
        …
    };
}

Person person = new Person("Fred Flintstone", 35);
