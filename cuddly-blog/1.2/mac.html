Mac

  1. OS X 下卸载 pkg 包 

# Trash replace rm with mv, use trm to run true rm
alias trm='/bin/rm'
trash(){
  mv $@ /Users/alex/.Trash
}
alias rm='trash'

# 修改终端显示方式(centos)
export PS1="[\u@\h \W]\$"

# 设置JAVA_HOME
JAVA_HOME=$(/usr/libexec/java_home -v 1.7)

# 禁止生成"._"文件 http://www.eifr.com/article.php?id=1643&act=print
vi .bashrc
# disable special creation/extraction of ._* files by tar, etc. on Mac OS X
COPYFILE_DISABLE=1;
export COPYFILE_DISABLE


# brew 装机软件
proxychains-ng	wget    python		tmux    jq  unrar

# 清理微信空间
\rm -rf ~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application\ Support/com.tencent.xinWeChat/*



# 关闭摄像头
sudo chmod a-r /System/Library/Frameworks/CoreMediaIO.framework/Versions/A/Resources/VDC.plugin/Contents/MacOS/VDC

#开启摄像头
sudo chmod a+r /System/Library/Frameworks/CoreMediaIO.framework/Versions/A/Resources/VDC.plugin/Contents/MacOS/VDC

# 获取 DNS
networksetup -getdnsservers Wi-Fi

#清理 DNS 缓存
sudo discoveryutil mdnsflushcache

# 设置 DNS
networksetup -setdnsservers Wi-Fi 8.8.8.8 8.8.4.4

Maven 随记

  1. 安装jar包到本地仓库

     mvn install:install-file -Dfile=./lib/jackson-all-1.7.6.jar -DgroupId=org.codehaus.jackson -DartifactId=jackson-all -Dversion=1.7.6 -Dpackaging=jar

  2. 提取maven仓库中的jar包

    • mvn dependency:copy-dependencies -DoutputDirectory=lib -DincludeScope=compile

  3. Maven常用命令

    • mvn archetype:create # 创建 Maven 项目

    • mvn compile # 编译源代码

    • mvn test-compile # 编译测试代码

    • mvn test # 运行应用程序中的单元测试

    • mvn site # 生成项目相关信息的网站

    • mvn clean # 清除目标目录中的生成结果

    • mvn package # 依据项目生成 jar 文件

    • mvn install # 在本地 Repository 中安装 jar

    • mvn eclipse:eclipse # 生成 Eclipse 项目文件

    • mvn eclipse:eclipse –Dwtpversion=1.0 # 生成Eclipse WTP 项目

    • mvn package -DskipTests # mvn编译跳过单元测试

  4. Maven内置变量说明:

    ${basedir} 项目根目录
${project.build.directory} 构建目录，缺省为target
${project.build.outputDirectory} 构建过程输出目录，缺省为target/classes
${project.build.finalName} 产出物名称，缺省为${project.artifactId}-${project.version}
${project.packaging} 打包类型，缺省为jar
${project.xxx} 当前pom文件的任意节点的内容
