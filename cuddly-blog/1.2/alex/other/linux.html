Linux 相关杂记

SSH 无密

  1. 步骤1:用 ssh-key-gen 在本地主机上创建公钥和密钥

    • $ ssh-keygen -t rsa

  2. 步骤2:用 ssh-copy-id 把公钥复制到远程主机上

    • $ ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.0.3

    • 注: ssh-copy-id 把密钥追加到远程主机的 .ssh/authorized_key 上.

  3. 步骤3:直接登录远程主机

    • $ ssh remote-host

Mac ssh无密登陆远程linux

mac$ cat id_rsa.pub >> authorized_keys
mac$ cd ~/.ssh ; scp -r authorized_keys linux:~/.ssh
linux$ chmod 755 ~
linux$ mkdir -p ~/.ssh
linux$ chmod 700 ~/.ssh
linux$ chmod 600 ~/.ssh/authorized_keys

命令随记

  1. awk累加

    • lsof -s | grep mem | awk '{sum += $7} END {print sum}'

  2. incrontab 文件监控

  3. Wget下载整个目录

    • wget -c -r -np -k -L -p URL

  4. 杀死僵尸进程

    • 僵尸进程就是进程状态是Z

      ps axjf 找出僵尸进程的父进程. kill -9 父进程即可

    • 杀死进程

      ps -ef | grep defunct #找出僵尸进程
ps -e -o ppid,stat | grep Z  #找出僵尸进程的父进程
ps -e -o ppid,stat | grep Z | cut -d" " -f2 | xargs kill -9  # 批量杀死

  5. 修改隐藏属性

    • chattr -ia /etc/resolv.conf

    • chattr +i /etc/resolv.conf

    • chattr +a /etc/resolv.conf

    • lsattr /etc/resolv.conf

磁盘相关

  1. U盘格式化

    • sudo fdisk -l

    • umount /dev/sdb

    • mkfs.vfat /dev/sdb

  2. 硬盘格式化

    • fdisk /dev/sdc #对硬盘进行分区,依次输入 n, p, 1, 以及两次回车，然后是 wq，完成保存

    • fdisk -l #即可查看分区

    • mkfs -t ext4 /dev/sdc #分区之后格式化硬盘

    • mount -t ext4 /dev/sdc /mnt/sdc #然后挂载硬盘

  3. linux挂载硬盘

    • 如果使用 UUID，则先通过 blkid 命令，得到磁盘的 UUID.

    • blkid /dev/sdc

      • /dev/sdc: UUID="185dc58b-3f12-4e90-952e-7acfa3e0b6fb" TYPE="ext4"

    • vi /etc/fstab #加入UUID=185dc58b-3f12-4e90-952e-7acfa3e0b6fb

    • /mnt/mydisk ext4 defaults 0 2 UUID=5c51db3c-f45f-467c-86fb-3e4453ddfbb8 /opt ext4 defaults 0 2

  4. 制作u盘启动

    • dd if=~/Fedora-18-x86_64-DVD.iso of=/dev/sdb

    • dd if=你的iso路径of=你的u盘设备名称

    • sudo usb-creator-gtk #ubuntu下制作u盘启动

  5. Mac 制作 U 盘启动

    $ diskutil list
...

# if mounted somewhere
$ sudo diskutil unmount $device

# all the partitions (there's also a "force" option, see the manual)
$ sudo diskutil unmountDisk $device

# remember zip drives? this would launch them. good times!
$ sudo diskutil eject $device

# formate disk
$  diskutil eraseDisk ExFAT alex disk2

  6. 挂载本地iso文件

    • `mount -o loop CentOS-6.4-x86_64-bin.iso /media/ `

  7. 远程挂载

    • mount -t cifs -o username=win7Username,password=123456,rwo //192.168.3.229/share /mnt/share/ #linux挂载远程硬盘,挂载win7共享目录

配置Centos Yum源

本地源

  1. 挂载本地iso

    • mount -o loop CentOS-6.4-x86_64-bin.iso /media/ #挂载本地iso文件

  2. 配置本地yum源

cd /etc/yum.repos.d/   #进入yum配置目录
vi  CentOS-Media.repo
[GuangPan]
name=CentOS-$releasever - Media
baseurl=file:///media/
gpgcheck=0
enabled=1

ELRepo源

  1. 导入public key

    • rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org

  2. 安装ELRepo到CentOS-6.5中

    • rpm -ivh http://www.elrepo.org/elrepo-release-6-5.el6.elrepo.noarch.rpm

Nginx 代理源

  1. 背景

    • 局域网内,只有一台服务器可以上网.

    • 使用代理配置其他不能联网的机器也可以使用yum

  2. Nginx配置文件

# mirrors.conf
 server
    {
        listen 80;
        #listen [::]:80;
        server_name mirrors.hadoop1.com;
        index index.html index.htm index.php default.html default.htm default.php;
        root  /usr/share/nginx/html;

        location /ubuntu/ {
            proxy_pass http://mirrors.aliyun.com/ubuntu/ ;
        }

        location /centos/ {
            proxy_pass http://mirrors.aliyun.com/centos/ ;
        }

        location /centosorg/ {
            proxy_pass http://mirror.centos.org/centos/ ;
        }

        location /epel/ {
            proxy_pass http://mirrors.aliyun.com/epel/ ;
        }
}
