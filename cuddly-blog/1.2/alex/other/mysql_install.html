mysql 安装

Centos

  1. yum -y install mysql-server

  2. service mysqld start

install mysql5.6

  1. 下载repo:wget http://repo.mysql.com/mysql-community-release-el6-5.noarch.rpm

  2. 安装这个repo:yum localinstall mysql-community-release-el6-5.noarch.rpm

  3. 查看repo是否被添加: yum repolist enabled | grep "mysql.-community."

  4. 按住mysql:yum install mysql-community-server

  5. 初始化db:mysql_install_db --user=mysql

  6. 启动mysql:service mysqld start

  7. 开启远程:update user set host = '%' where user = 'root' and host = 'localhost';

  8. flush privileges;

homebrew安装

初始化

sudo mysql_install_db --verbose --user=`whoami` --basedir="$(brew --prefix mysql)" --datadir=/usr/local/var/mysql --tmpdir=/tmp --explicit_defaults_for_timestamp

  1. 启动

    • /usr/local/opt/mysql/bin/mysqld_safe &

  2. 停止

    • mysqladmin -uroot -proot shutdown

  3. 屏蔽权限启动

    • mysqld_safe --skip-grant-table &

    • mysql -u root mysql

    • UPDATE user SET Password=PASSWORD('newpassword') where USER='root';

    • FLUSH PRIVILEGES;

  4. 修改密码

    • /usr/local/opt/mysql/bin/mysqladmin -u root password 'new-password'

    • /usr/local/opt/mysql/bin/mysql_secure_installation

    • mysql -u root -p

新增用户及授权

  • mysql 新增用户及授权

SELECT user.Host,user.User,user.Password FROM user  order by user ;
delete from user where user='idfa' and host = 'localhost'

  • 新建用户,授权只读(多个host执行多次)

insert into mysql.user(Host,User,Password) values("192.168.1.111","idfa",password("B2nwTD6xGd"));
grant select on mobile.ipad_idfa to idfa@192.168.1.111 ;
grant select on mobile.iphone_idfa to idfa@192.168.1.111 ;

  • 撤销所有授权

revoke all on mobile.ipad_idfa from idfa@192.168.1.111 ;
revoke all on mobile.iphone_idfa from idfa@192.168.1.111 ;
flush privileges;
